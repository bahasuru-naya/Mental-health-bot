# Mental-health-bot
RASA chatbot to assist mental health

# To run  chatbot in terminal
## run follwing command in terminal 
```
rasa shell
 ```
```
rasa run actions 
 ```

# To run  chatbot in GUI application(socketio)
## run follwing command in terminal 
```
rasa run -m models --enable-api --cors "*"
 ```
```
rasa run actions 
 ```
